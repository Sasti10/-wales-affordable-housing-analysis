# Wales Affordable Housing Analysis 2019–2025

## Overview
Analysis of affordable housing delivery across Wales using real 
government data from StatsWales.

## What I Did
- Downloaded real Welsh Government housing data
- Cleaned 11,412 rows using Python and Pandas
- Analysed delivery trends across 22 local authorities
- Built an interactive Power BI dashboard

## Key Findings
- Cardiff leads Wales with 19,452 units delivered
- Delivery dipped in 2021-22 due to COVID but recovered 
  to record highs in 2024-25
- Registered Social Landlords deliver 4x more than 
  Local Authorities
- 74% of housing is Capital Grant funded

## Tools Used
- Python — Pandas
- Power BI — DAX measures
- StatsWales — Data source
